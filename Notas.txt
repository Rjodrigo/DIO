*Código para criação do archetype: mvn archetype:generate -DgroupId=one.digitalinnovation -DartifactId=quick-start-maven -Darchetype=maven-archetyoe-quickstart -Dinteractive Mode=false

*comando para compilar as classes: mvn compile

* Comando para testar: mvn test

*Comando para enpacotar: mvn package

*Comando para limpar a pastas do arquivo target(teste): mvn clean

*Comando para publicar (localmente) o seu projeto para outra pessoas utilizarem: mvn install
